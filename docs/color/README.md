### HSB = HSV, HSL


HSB = 色相  饱和度  明度,  饱和度就是白色混入的量   越少就越纯，就越饱和，直到没有白色  明度就是黑色混入的量  越少就越明亮 直到没有黑色

HSL = 色相 饱和度 明度  饱和度是灰色混入量的描述也就是纯度  明度就是明暗变化 黑白
